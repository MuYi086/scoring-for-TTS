# 中文转写规范化：zh-v1

此规则只用于把冻结参考文本与 ASR 原始转写变成可比较的字符串；不会改写原始转写，也不会改变音频。

1. 对两侧文本同时应用 Unicode NFKC 兼容规范化。
2. 英文字母统一为小写；连续空白折叠后移除，以中文字符序列计算 CER。
3. 删除中英文常用标点符号，但保留汉字、拉丁字母、阿拉伯数字和其他有语义的字符。
4. 默认不把阿拉伯数字改写为中文读法，也不展开缩写、专名或单位。若某个样本必须采用读法转换，应在冻结清单的评测说明中声明，并对所有候选模型使用相同转换。
5. 以 Levenshtein 编辑距离除以规范化参考文本的字符数得到 CER；插入可能使该值大于 1，应如实保留，不截断。

规则版本改变时必须新建文件和新的 `profile_id`，不得覆盖历史结果。
