# 有声书制作 TTS 综合打分排行榜

## 生产结论

**建议优先使用 Qwen3-TTS-12Hz-1.7B-Base 进入有声书生产试运行。**

它在 V2、V3 两批评测的 SenseVoice CER 与 Whisper CER 四个文本正确性排名中全部第一，同时两批 UTMOSv2 自然度排名也全部第一。对长篇有声书而言，少读、错读和同音误读会直接放大人工校对、切句重做与后期拼接成本，因此这项稳定优势比单次自然度或单个音色后端领先更有生产价值。

Qwen3-TTS 的主要风险是 SpeechBrain ECAPA 音色相似度在两批中均为第 7，不能据此声称它是“最像参考音频”的模型。正式全量生产前，应让 Qwen3-TTS 与第二名 LongCat-AudioDiT-1B 使用同一批角色参考音频完成一次人工盲听；默认生产主线仍建议从 Qwen3-TTS 开始。

## 评分方法

综合榜使用 V2、V3 六份评价报告中的 12 个独立排名：

- 文本正确性 40%：SenseVoice CER、Whisper CER，各自包含 V2 与 V3；
- 音色相似度 35%：WavLM SIM、SpeechBrain ECAPA SIM，各自包含 V2 与 V3；
- 自然度 25%：UTMOSv2、NISQA-TTS，各自包含 V2 与 V3。

不同后端的原始值不共用量纲，因此不直接相加。每个后端只将名次转换为百分位分：`100 × (8 - 名次) / 7`，第一名为 100 分，第八名为 0 分；同一维度内四个后端/批次等权，再按 40% / 35% / 25% 形成生产优先分。

文本正确性权重最高，是因为长篇内容中的错读会持续制造返工；音色相似度紧随其后，因为角色声线是声音克隆的核心；自然度权重略低，是因为 UTMOSv2 和 NISQA-TTS 不评价情绪表演、角色塑造和长程韵律，仍需人工盲听补充。

## 为什么榜首是 Qwen3-TTS

| 生产问题 | Qwen3-TTS 证据 | 影响 |
| --- | --- | --- |
| 台词能否稳定念对 | V2/V3 的 SenseVoice、Whisper 四项排名全部第 1 | 长篇校对和重生成成本最低，构成最主要优势。 |
| 声音是否自然 | UTMOSv2 两批第 1，NISQA-TTS 为第 3/第 2 | 两套自然度后端整体都处于第一梯队。 |
| 是否像参考音频 | WavLM 为第 6/第 3，ECAPA 为第 7/第 7 | 这是明确短板，必须用真实目标角色做人工盲听门禁。 |
| 跨批次是否稳定 | 文本与自然度领先在 V2、V3 重复出现 | 相比只在单批次领先的模型，生产风险更可控。 |

LongCat 是最重要的对照候选：它的 WavLM 为第 2/第 1，ECAPA 为第 1/第 4，自然度也位于前列。如果人工盲听确认 Qwen3-TTS 的音色偏差不可接受，应把 LongCat 升为主模型；但在此之前，LongCat 的 V2 CER 风险不足以支持直接取代 Qwen3-TTS。

## 上线前门禁

当前报告没有评价长篇连续稳定性、情绪表演、停顿节奏、专名读音、生成吞吐、显存成本、许可范围或云端费用。进入整本书生产前，建议先制作 30–60 分钟试播集：

1. 同时使用 Qwen3-TTS 与 LongCat，覆盖旁白、主要男性角色、主要女性角色和高情绪对白；
2. 逐句记录错读率、漏字率、重生成次数和人工修复分钟数；
3. 对音色相似、自然度、角色感和长程稳定性进行隐藏模型名的人工盲听；
4. 若 Qwen3-TTS 音色达到可接受线，直接保留榜首进入生产；若未达到而 LongCat 的人工错读率可控，再切换 LongCat。
