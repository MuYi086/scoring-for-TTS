# utmosv2：合成语音自然度预测

UTMOSv2 用于为每条已合成音频预测自然度 MOS（平均主观意见分）。它是自动评价栈的一个维度，不评估文本是否读对、目标音色是否贴合，也不判断停顿、语气和情绪是否符合个人创作意图。

自动入口会把它与 WavLM、ASR/CER、削波检查和长程音色稳定度一起运行。结果字段为 `utmosv2.predicted_mos`，数值越高代表模型预测的自然度越高；仅可在同一 UTMOSv2 版本、相同样本集内比较。

## 采用原因与限制

- 官方实现面向高质量合成语音的自然度 MOS 预测，支持按 WAV 文件或目录推理。
- 它不是 7B 音频语言模型，适合与 16 GB 显存机器上的 ASR、WavLM 串行执行。
- 预训练分数不是中文 TTS 的通用合格线。必须先在 `calibration` 样本上确认它能把真人干净音频、削波音频、错误语速或明显损坏音频按合理方向区分。
- 不能把 UTMOSv2 分数解释为你个人对停顿、语气或情绪的偏好；这些保留在 `listener-review/` 的独立记录中。

实现、依赖和模型行为以 [UTMOSv2 官方仓库](https://github.com/sarulab-speech/UTMOSv2) 为准。
